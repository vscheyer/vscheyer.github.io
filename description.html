How to use FormFit

Attach one accelerometer to your ankle and the other accelerometer to your lower back as indicated in the picture below. Start the data collection, and then run as usual. Load the data into the Matlab program.

How to read the results

For pelvic tilt, a zero means you don't have too much pelvic tilt. This is a good thing. If the result reads one, try to run with your body in a more upright position.

Stride frequency outputs a number which tells you how many strides you take per minute. Try to aim for 180 strides per minute.

For heel strike, a zero means you are not striking the ground too hard. This is good. If the result reads one, try to run more on the balls of your feet with less impact.
